# Roadmap

## Current Version (0.1.0)

### Completed

- [x] FastAPI backend with core endpoints
- [x] IBM RXN integration via rxn4chemistry
- [x] Deterministic placeholder procedure generation
- [x] Risk annotation based on lab context
- [x] Feedback collection system
- [x] Basic Next.js frontend
- [x] Docker support
- [x] Unit tests

- [ ] Rate limiting
- [ ] Request logging and analytics
- [ ] Frontend improvements (better UX)
